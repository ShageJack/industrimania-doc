# Industrimania <small>工业狂热</small>
> Industrimania（工业狂热）是为模组包Forsaken Odyssey（遗世流浪）制作的一个独立模组。它旨在给游戏加入更多现实风格的内容，并试图创造一套更符合人类科学技术发展史的进度系统。
